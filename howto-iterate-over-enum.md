#### e) Enum
- **Question:** How to iterate over an Enum's values?

- **Explanation:** Enums are a type-safe way to represent constant values.
- **Sample Code:**

  ```java
  enum Color { RED, GREEN, BLUE }
  for (Color c : Color.values()) {
    System.out.println(c);
  }
  ```
